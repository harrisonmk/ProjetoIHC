DESEMPENHO DO JAVA

O desempenho de um programa Java compilado por bytecode Java depende de como suas tarefas são gerenciadas de maneira ideal pela JVM ( Java Virtual Machine ) do host e como a JVM explora os recursos do hardware e sistema operacional (SO) do computador ao fazer isso. Assim, qualquer teste de desempenho ou comparação de Java deve sempre relatar a versão, o fornecedor, o SO e a arquitetura de hardware da JVM usada. De maneira semelhante, o desempenho do programa nativamente compilado equivalente dependerá da qualidade de seu código de máquina gerado, portanto, o teste ou comparação também deve relatar o nome, a versão e o fornecedor do compilador usado e suas diretivas de otimização de compilador ativadas. .
